  • Change the color

  • Resize to a size so small that it compromises legibility

Buttons

"Call to action in web design — and in user experience (UX) in particular — is a term used for elements in a web page that solicit an action from the user. The most popular manifestation of call to action in web interfaces comes in the form of clickable buttons that when clicked, perform an action (e.g. "Buy this now!") or lead to a web page with additional information (e.g. "Learn more…") that asks the user to take action."

Jacob Gube | Six Revisions | Read Article
Primary call-to-action button

Call-to-action buttons should be flat, rectangular, and in most cases stick with the OS BLUE [#0093DB] palette. Words within the rectangle should be white, uppercase and utilize OneScreen's fonts.

Secondary call-to-action button

Secondary call-to-action buttons utilize the gray palette and should be used for less important links.

The 16x9 Ratio

"16:9 is an aspect ratio with a width of 16 units and height of 9. Since 2009, it has become the most common aspect ratio for sold televisions and computer monitors and is also the international standard format of HDTV, Full HD, non-HD digital television and analog widescreen television."

OneScreen uses the 16x9 aspect ratio as an element of the brand to emphasize our mission. That mission is to shape the new era of television by making it easy for you to watch video through any screen at any time.

Letterhead

With the power of email, it's not everyday that letters get printed and sent out via snail mail here at OneScreen but just in case you need it, here are some tips for writing a consistent letter:

  • Margins are 1" all around
  • Use DINPro-Regular for the body copy. You can use any font you like but ideally choose a sans-serif font such as Source Sans Pro, Helvetica, or Arial.
  • Font size should be 12pt
  • Left align all copy
  • Line spacing should be at 1.15
  • Text color should be black
